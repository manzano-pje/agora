# 🏛️ Sistema de Gestão de Associações - Agora

Este sistema tem como objetivo oferecer uma solução completa para o gerenciamento de associações, permitindo o controle de associados, projetos, participação em projetos, diretoria (atual e anteriores), financeiro e emissão de relatórios personalizados.

## ✨ Funcionalidades Gerais

- Cadastro e gerenciamento de **Associados**
- Registro e acompanhamento de **Projetos**
- Atribuição de **Associados a Projetos**
- Cadastro de **Diretoria atual e anteriores**
- Controle **Financeiro** (em definição)
- Geração de **Relatórios** diversos:
  - Associados cadastrados
  - Projetos por período
  - Projetos por associado
  - Diretoria atual e anteriores
  - Relatórios financeiros
